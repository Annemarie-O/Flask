## Set Up

Ensure that the latest version of Python is installed.

1. Run the following command in termninal
```bash
git clone https://github.com/The-Athenian/Flask
```

2. Run the following commands in terminal to install required scripts
```bash
pip install flask
pip install flask-sqlalchemy
pip install flask-login
```

To run the app:
1. Run `main.py`
```bash
python main.py
```
2. Go to http://127.0.0.1:5000 to access the website.
